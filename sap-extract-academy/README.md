# SAP Extract Academy

Learn SAP S/4HANA data extraction the right way — straight from SAP's documentation.

A free, interactive learning site that teaches data engineers, architects, and SAP Basis professionals how to extract data from S/4HANA and ECC into cloud platforms.

**Launch Stage:** S/4HANA on-prem + ECC 6.0  
**Tables:** VBAK, LFA1, MARA, BKPF, ACDOCA

Visit the [project documentation](../academy_handoff/) for complete specifications and development guidelines.
