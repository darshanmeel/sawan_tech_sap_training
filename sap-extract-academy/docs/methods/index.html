OData

OData / ODP via CDS →

The default for S/4HANA. Runtime-license compatible, scales with partitioning.

RFC/BAPI

RFC / BAPI →

Works on anything. Fast. Requires a real SAP RFC user and careful row-size handling.

Direct DB

Direct DB read →

JDBC straight to HANA or the underlying DB. Fastest path, highest license risk.

BW Open Hub

BW Open Hub →

BW-mediated extraction via Open Hub destinations. Safe license posture if BW is already in the landscape.

Flat File

Flat file export →

Scheduled ABAP jobs writing CSV/Parquet to shared storage. The boring, unkillable option.
