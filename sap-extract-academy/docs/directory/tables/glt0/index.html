What this is

Classic general-ledger totals table in ECC — aggregated balances by account, company code, and fiscal period. Replaced by ACDOCA in S/4HANA.

GLT0 is the classic general-ledger totals table in SAP ECC. It stores pre-aggregated balances by G/L account, company code, fiscal year, and period — the table that classic balance-sheet and trial-balance reports read from.

In SAP S/4HANA, GLT0 no longer exists as a persistent table. The Universal Journal (ACDOCA) carries every posting at line-item granularity, and aggregations (trial balance, balance sheet totals) are computed on the fly from ACDOCA via the released finance CDS views.

This is a stub page: its purpose is to resolve the cross-mode redirect from ACDOCA's ECC equivalents list. Full curation (columns, extract methods, notes, ingestion) is out of scope for directory v1.
