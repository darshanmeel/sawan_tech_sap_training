What this is

In ECC, the same shape and role; MATNR length is 18 characters rather than S/4's extended 40.

MARA is the material master header — one row per material, keyed by (MANDT, MATNR). Every downstream MM, SD, and PP table eventually joins through here, which makes MARA both small in row count and structurally critical in a warehouse.

The single biggest extraction gotcha is MATNR width. In ECC, MATNR is 18 characters. In S/4HANA, the extended format is 40 characters. Pipelines that feed both systems into the same warehouse must standardize on 40 at ingestion; silent truncation on the 18-char side is the most common data-quality bug on the material master, and it surfaces only after you start creating materials with longer numbers post-S/4 migration.

Scope lock

MATNR is 18 characters; many downstream systems have hard-coded that width. Plan early if you intend to move to S/4 and adopt the 40-character shape.

Generated from the column schema, adjusted for each warehouse's type system and partitioning model. Dialect quirks noted inline.

-- Snowflake · MARA · generated from column schema
CREATE OR REPLACE TABLE SAP_MM.MARA (
  MANDT     VARCHAR(3) NOT NULL,
  MATNR     VARCHAR(18) NOT NULL,
  ERSDA     DATE,
  ERNAM     VARCHAR(12),
  MTART     VARCHAR(4),
  MATKL     VARCHAR(9),
  MEINS     VARCHAR(3),
  BRGEW     NUMBER(13, 3),
  NTGEW     NUMBER(13, 3),
  VOLUM     NUMBER(13, 3),
  MBRSH     VARCHAR(1),
  MSTAE     VARCHAR(2),
  LVORM     VARCHAR(1),
  CONSTRAINT PK_MARA PRIMARY KEY (MANDT, MATNR)
)
CLUSTER BY (MANDT, MATNR);

Snowflake ingestion. Same daily-snapshot pattern; just keep the target MATNR column at 40 chars from day one so ECC and S/4 land in the same dimension cleanly.

-- Databricks · MARA · Delta Lake
CREATE OR REPLACE TABLE sap_mm.mara (
  mandt     STRING NOT NULL,
  matnr     STRING NOT NULL,
  ersda     DATE,
  ernam     STRING,
  mtart     STRING,
  matkl     STRING,
  meins     STRING,
  brgew     DECIMAL(13, 3),
  ntgew     DECIMAL(13, 3),
  volum     DECIMAL(13, 3),
  mbrsh     STRING,
  mstae     STRING,
  lvorm     STRING
)
USING DELTA
PARTITIONED BY (mandt, matnr)
TBLPROPERTIES ('delta.autoOptimize.optimizeWrite' = 'true');

Databricks ingestion. Auto Loader to Delta; normalize ECC MATNR to 40 chars (right-pad) in the ingestion job.
