Real-Time CDC

SLT - SAP Landscape Transformation

Trigger-based real-time replication with <1 minute delta lag. Ideal for on-premise to on-premise data movement with complex transformations.

Key Features:
  • Real-time delta replication
  • Trigger-based CDC
  • Multiple target options
  • Recommended for HANA targets
SLT How-To Guide →
Modern Framework

ODP - Operational Data Provisioning

Modern SAP framework with multi-subscriber support. Best for external analytics tools with 15-60 minute delta latency tolerance.

Key Features:
  • Framework-based approach
  • Multi-subscriber capability
  • OData Service exposure
  • Cloud-friendly integration
ODP How-To Guide →
Legacy Support

RFC - Remote Function Call

Traditional SAP protocol for internal system communication. Still used for custom ABAP-to-ABAP integration (restricted for third-party use by SAP).

Key Features:
  • Internal SAP-to-SAP
  • Custom ABAP integration
  • Legacy system support
  • Simple setup
RFC How-To Guide →
REST-Based

OData Services Integration

HTTP/HTTPS based REST integration. Perfect for cloud platforms and analytics tools with firewall-friendly connectivity needs.

Key Features:
  • REST/HTTP protocol
  • Firewall-friendly
  • JSON format support
  • Cloud platform ready

Cloud-Native

SAP Datasphere Replication Flows

Cloud-native managed service with automatic CDC via Resilient Data Buffer. Ideal for long-term cloud migration strategy.

Key Features:
  • Cloud-native architecture
  • Managed service
  • Automatic CDC (RDB)
  • Cloud target focused
